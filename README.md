# Student Notes App

Jednostavna Python aplikacija za rad sa studentskim beleškama.

## Funkcionalnosti

- Dodavanje beleške
- Prikaz svih beleški
- Prikaz rednog broja uz svaku belešku
- Izlaz iz programa

## Pokretanje aplikacije

python main.py

## Svrha projekta

Projekat se koristi za učenje osnovnog rada sa Gitom i GitHub-om.
